# User Workflow Guide: From Master to Multi-Platform

This guide outlines the standard operating procedure for creative teams using OVERLORD.

## Step 1: Media Ingestion
- Navigate to the **Studio**.
- Drag and drop your "Master Content" (High-resolution video or image).
- Assign a **Project Name** and **Brand Kit** (defines logos, fonts, and color palettes).

## Step 2: Adaptation Configuration
Select the target platforms from the dashboard:
- **Vertical (9:16):** TikTok, IG Reels, YouTube Shorts.
- **Square (1:1):** Instagram Feed, LinkedIn.
- **Landscape (16:9):** YouTube, Twitter.
- **Text-Only:** Thread/Tweet summaries, LinkedIn Articles.

## Step 3: AI Review (Human-in-the-Loop)
- Open the **Live Preview**.
- View the "Saliency Map" generated by the Visionary Agent to ensure the subject stays in-frame.
- Review the **Tone Shift** suggested by the Copywriter Agent for platform-specific captions.

## Step 4: Refinement & Approval
- Adjust subtitle placement or edit AI-generated hooks using the Studio editor.
- Click **"Finalize Adaptation"** to trigger high-quality rendering.

## Step 5: Export & Distribution
- Access the **Asset Library**.
- Batch download all variants or use direct-to-platform API connectors to schedule posts.
